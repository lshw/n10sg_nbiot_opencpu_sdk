◆ opencpu_i2c_init()

void opencpu_i2c_init ( void  )

I2C初始化

Returns
空
示例：使用I2C接口进行数据读写
opencpu_i2c_init();
unsigned char buffer[2];
//向总线地址为0x77的外设写入0xD0，然后读出一个字节
opencpu_i2c_write_read(0x77,0xD0, buffer, 1);
//从总线地址为0x77的外设读出2个字节
opencpu_i2c_read(0x77,buffer, 2);
//向总线地址为0x77的外设写入2个字节
opencpu_i2c_write(0x77,buffer, 2);
//使用完毕后，调用I2C释放函数：
opencpu_i2c_deinit();
